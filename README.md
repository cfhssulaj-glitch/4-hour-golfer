# 4-Hour Golfer

A weekly golf practice and fitness schedule generator based on TPI (Titleist Performance Institute) methodology. Select your practice days, and the app generates randomized sessions with variety across golf drills and fitness exercises.

**Live at: https://4hourgolfer.vercel.app**

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Radix UI + Lucide icons
- PWA-enabled (installable on mobile)

## Development

```bash
npm install
npm run dev
```

## Deployment

Hosted on [Vercel](https://vercel.com) with automatic deploys from the `main` branch on GitHub. No backend — all data lives in localStorage per device.
